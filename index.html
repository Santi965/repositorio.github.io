Introducción

Bienvenido al curso de Desarrollo de Aplicaciones Web. Este curso está diseñado para proporcionarte las habilidades y conocimientos necesarios para crear aplicaciones web modernas y dinámicas. CONTENIDO

SEMANA 1

TECNOLOGÍAS WEB

Las tecnologías web son el conjunto de herramientas, lenguajes y protocolos que permiten la creación, desarrollo y funcionamiento de aplicaciones y sitios web. A continuación, se desglosan los componentes fundamentales de estas tecnologías.

SEMANA 2

ESTRUCTURA HTML Y XML

HTML (HyperText Markup Language) y XML (eXtensible Markup Language) son lenguajes de marcado utilizados para estructurar y transportar datos en la web. HTML se centra en la presentación de datos para su visualización en navegadores web, mientras que XML se utiliza principalmente para almacenar y transportar datos de manera estructurada y legible tanto para humanos como para máquinas.

SEMANA 3

FRONT-END CON LIBRERÍAS

En el panorama del desarrollo web front-end, las librerías de estilo como CSS, Bootstrap y Tailwind juegan roles cruciales al definir la apariencia y la interactividad de las páginas web. CSS, siendo el lenguaje fundamental de estilo, ofrece una flexibilidad total para la personalización de la presentación. Bootstrap, por otro lado, proporciona un conjunto completo de componentes y estilos predefinidos, lo que agiliza el proceso de desarrollo y garantiza una apariencia consistente en todas las páginas. Tailwind, con su enfoque basado en clases, permite una personalización detallada y modular, facilitando la creación de interfaces web altamente personalizadas.

SEMANA 4

FRONT-END CON JAVASCRIPT

JavaScript es un lenguaje de programación utilizado principalmente para crear contenido interactivo en páginas web. Es un componente esencial del desarrollo front-end, permitiendo la manipulación dinámica de elementos HTML, interacciones del usuario y comunicación con servidores.

SEMANA 5

NODE.JS

Node.js es un entorno de ejecución de JavaScript del lado del servidor que utiliza el motor V8 de Google Chrome. Permite a los desarrolladores crear aplicaciones web altamente escalables y de alto rendimiento utilizando JavaScript en el lado del servidor.

SEMANA 6

REACT

React es una biblioteca de JavaScript de código abierto utilizada para construir interfaces de usuario interactivas y reutilizables. Desarrollada por Facebook, React se destaca por su enfoque en la creación de componentes reutilizables y su eficiencia en la actualización del DOM.

SEMANA 10

HOOKS

Los Hooks son una característica introducida en React 16.8 que revolucionó la forma en que manejamos el estado y los efectos en componentes funcionales. Antes de los Hooks, necesitábamos usar clases para acceder al estado y a los ciclos de vida de React. Con los Hooks, podemos hacerlo directamente en componentes de función.

SEMANA 11

NEXT. JS

Next.js es un framework de React que ofrece optimizaciones incorporadas, enrutamiento avanzado y soporte CSS. Es ideal para construir aplicaciones web de alta calidad.

SEMANA 12

JavaServer Pages (JSP)

JSP es una tecnología basada en Java para crear páginas web dinámicas en el servidor. Permite separar la lógica de negocio del HTML generado.
