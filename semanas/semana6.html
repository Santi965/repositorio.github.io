SEM 6 - REACT

React es una biblioteca de JavaScript de código abierto utilizada para construir interfaces de usuario interactivas y reutilizables. Desarrollada por Facebook, React se destaca por su enfoque en la creación de componentes reutilizables y su eficiencia en la actualización del DOM.

Componentes

  • Componentes funcionales: Estos son componentes que se definen como funciones de JavaScript y se utilizan para representar la interfaz de usuario de manera más simple y concisa.
  • Componentes de clase: Estos son componentes que se definen como clases de JavaScript y tienen un estado interno y métodos de ciclo de vida que les permiten manejar la lógica y el estado de la interfaz de usuario.
  • Componentes de orden superior (HOC): Son componentes que toman un componente y retornan otro componente, agregando funcionalidades adicionales al componente original.
  • Componentes de renderización condicional: Permiten renderizar diferentes elementos o componentes en función de una condición específica, lo que proporciona flexibilidad en la presentación de la interfaz de usuario.

Principal Componente

El componente principal en React es el "componente". Un componente en React es una pieza de la interfaz de usuario que puede contener HTML, CSS y JavaScript, y puede ser reutilizado en múltiples partes de una aplicación.

Características

Virtual DOM

React utiliza un Virtual DOM para mejorar la eficiencia en la actualización del DOM. Esto significa que React compara los cambios en el estado de la aplicación con una representación virtual de la interfaz de usuario y solo actualiza los elementos que han cambiado, en lugar de volver a renderizar toda la página.

Reutilización de componentes

Fomenta la reutilización de componentes, lo que permite crear una interfaz de usuario consistente y fácil de mantener al dividir la interfaz en componentes más pequeños y reutilizables.

Unidireccionalidad de datos

React sigue el principio de "unidireccionalidad de datos", lo que significa que los datos fluyen en una sola dirección a través de la aplicación, lo que facilita la comprensión y el mantenimiento del estado de la aplicación.

Componentes controlados

En React, los componentes controlados son aquellos cuyo estado está controlado por React en lugar del DOM. Esto permite una manipulación más precisa y consistente de los datos en la aplicación.

React Hooks

Los Hooks son una característica introducida en React que permite a los desarrolladores usar el estado y otros aspectos de React en componentes funcionales, lo que simplifica el manejo del estado y la lógica de los componentes.

Renderizado del lado del servidor

React permite el renderizado del lado del servidor, lo que significa que las aplicaciones pueden renderizarse en el servidor y enviarse al navegador como HTML estático, lo que mejora la velocidad de carga y el SEO.

REFLEXIÓN

React ha revolucionado la forma en que se desarrollan interfaces de usuario en la web al proporcionar un enfoque modular y eficiente para la construcción de aplicaciones interactivas y reactivas. Su énfasis en la reutilización de componentes, la eficiencia en la actualización del DOM y la unidireccionalidad de datos lo convierten en una opción poderosa y popular para el desarrollo web moderno. Y tres cosas que no debes olvidar es:

Los Componentes reutilizables en React

Se pueden utilizar en múltiples lugares dentro de una aplicación

La Eficiencia del Virtual DOM

Permite a React actualizar solo las partes del DOM real que han cambiado en lugar de volver a renderizar toda la página

Uso de React Hooks

Para simplificar el manejo del estado y la lógica de los componentes.

CONCLUSIÓN

React ha cambiado la forma en que los desarrolladores construyen aplicaciones web al proporcionar un enfoque modular, eficiente y reutilizable para la creación de interfaces de usuario interactivas y dinámicas. Su popularidad y su gran comunidad de desarrolladores lo convierten en una herramienta invaluable para el desarrollo web moderno.
